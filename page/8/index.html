Knative RoutingвҖӢ

KnativeлҠ” м•һм—җм„ңлҸ„ лӘҮлІҲ м–ёкёүн•ҳмҳҖм§Җл§Ң кё°ліём ҒмңјлЎң Routingмқ„ мӮ¬мҡ©н•ҳм—¬ мҷёл¶Җм—җ л…ём¶ңн•  м„ңл№„мҠӨл“Өм—җ лҢҖн•ң HTTP EndpointлҘј м ңкіөн•ңлӢӨ. м–ҙл–»кІҢ ліҙл©ҙ кё°ліём ҒмңјлЎң API Gateway м—ӯн• мқ„ н•ҳкё°лҸ„ н•ҳкі  Ingress м—ӯн• мқ„ н•ҳкё°лҸ„ н•ңлӢӨ. ліҙнҶө Service meshмқё IstioлҘј мӮ¬мҡ©н•ҳм—¬ ingressлҘј кө¬нҳ„н•ҳлҠ”кІғмқҙ лӢ№м—°н•ҳлӢӨкі  мғқк°Ғн•ҳкё°лҸ„ н•ҳм§Җл§Ң Istioмқҳ лӘЁл“  кё°лҠҘмқҙ Knativeм—җ н•„мҡ”н•ҳм§ҖлҠ” м•Ҡкі  м„Өм№ҳлҗҳлҠ”кІғ мһҗмІҙк°Җ лҰ¬мҶҢмҠӨ мҶҢлӘЁк°Җ кҪӨ лҗңлӢӨлҠ”кІғмқҖ м„Өм№ҳ н•ҙліёмӮ¬лһҢмқҖ м•Ңкі  мһҲмқ„кІғмқҙлӢӨ.

Kubernetesм—җм„ңлҠ” мқјл°ҳм ҒмңјлЎң м„ңл№„мҠӨ м ‘мҶҚмқ„ кө¬нҳ„н•ҳкІҢ лҗҳл©ҙ кё°ліём ҒмңјлЎң PodмҷҖ ServiceлҘј мғқм„ұн•ҳкі  IngressлҘј мӮ¬мҡ©н•ҳм—¬ нҒҙлҹ¬мҠӨн„° лӮҙл¶ҖлЎң л“Өм–ҙмҳӨлҠ” нҠёлһҳн”Ҫмқ„ мІҳлҰ¬н•ҳкІҢ лҗңлӢӨ.

KnativeвҖӢ


мқҙлҜём§Җм¶ңмІҳ : https://blog.openshift.com/knative-serving-your-serverless-services/

Knativeм—җм„ңлҠ” м•һм„  Knative кҙҖл Ё нҸ¬мҠӨнҢ…м—җм„ңлҸ„ м„ӨлӘ…н–Ҳл“Ҝмқҙ Automatic scaling up and down to zero нҠ№м„ұмқ„ к°Җм§Җкі  мһҲкё°м—җ Podк°Җ мөңмҙҲ мӢӨн–үлҗҳм–ҙмһҲм§Җ м•ҠмқҖ мғҒнғңм—җм„ң нҠёлһҳн”Ҫмқҙ л“Өм–ҙмҳӨкІҢ лҗҳл©ҙ Knative Serving Activatorм—җ мқҳн•ҙм„ң Podк°Җ м—ҶлҠ” Revisionмқ„ нҷ•мқён•ҳкі  Cold Start нҳ•нғңлЎң н”„лЎңл№„м ҖлӢқн•ҳкІҢ лҗңлӢӨ. лӮҳлҠ” мқҙкІҢ м§„м •н•ң м„ңлІ„лҰ¬мҠӨлқјкі  мғқк°Ғн•ҳм§Җл§Ң мЈјліҖм—җ л°ҳл°•н•ҳмӢңлҠ” л¶„л“ӨлҸ„ к°„нҳ№ мһҲлӢӨ.

мқҙнӣ„ Podк°Җ Warm мғҒнғңк°Җ лҗҳкі  лӮҳл©ҙ Istio Route(Ingress Gateway)лҘј нҶөн•ҙ нҠёлһҳн”Ҫмқҙ PodлЎң м „лӢ¬лҗҳм–ҙ нҶөмӢ мқҙ мқҙлӨ„м§ҖкІҢ лҗңлӢӨ.

нҳ„мһ¬ KnativeлҠ” нҳ„мһ¬ Ingress Gateway мқҳмЎҙм„ұмқ„ к°Җм§Җкі  мһҲкі  Envoyкё°л°ҳ Service Meshмқё Istio, Envoyкё°л°ҳ API Gatewayмқё Gloo л‘җк°Җм§Җ мҳөм…ҳмңјлЎң Ingress кө¬нҳ„мқҙ к°ҖлҠҘн•ҳлӢӨ.

IstioвҖӢ

KnativeлҠ” кё°ліём ҒмңјлЎң Ingress Gatewayкё°лҠҘмқ„ нғ‘мһ¬н•ҳкі  мһҲлҠ”лҚ° мқҙлҠ” Istioмқҳ кё°лҠҘмӨ‘ н•ҳлӮҳлӢӨ.
IstioлҠ” лӢӨмқҢкіј к°ҷмқҖ Core FeatureлҘј к°Җм§„лӢӨ. мғҒм„ён•ң лӮҙмҡ©мқҖ https://istio.io/docs/concepts/what-is-istio/ м—җм„ң нҷ•мқён•ҳл©ҙ лҗңлӢӨ.

  • Traffic management
  • Security
  • Policies and Telemetry
  • Performance and Scalability

IstioлҠ” 48к°ңмқҳ CRDs(CustomResourceDefinition objects)лҘј к°Җм§Җкі  мһҲлҠ”лҚ° мқҙмӨ‘ Knative Servingм—җм„ң мӮ¬мҡ©н•ҳлҠ”кұҙ VirtualService лӢЁ н•ҳлӮҳлӢӨ.

GlooвҖӢ

GlooлҠ” Kubernetes-native ingress controllerмқҙмһҗ Next Generation API Gateway лҘј мң„н•ҙ мӢңмһ‘лҗң н”„лЎңм қнҠёмқҙлӢӨ. мӢӨм ң Redhatм—җм„ң Openshiftкё°л°ҳ Microservice л°Ҹ Istio к°ңл°ңм—…л¬ҙлҘј н•ҳлӢӨк°Җ мөңк·јм—җ solo.ioмқҳ CTOлЎң мқҙм§Ғн•ң Christian Postaк°Җ л°Җкі  мһҲлҠ” н”„лЎңм қнҠёмқҙкё°лҸ„ н•ҳлӢӨ.

GlooлҠ” Envoy Proxy кё°л°ҳмңјлЎң лҸҷмһ‘н•ҳл©° кё°мЎҙ Legacyл¶Җн„° Containerм„ңл№„мҠӨ, FaaS(AWS Lambda, Azure Functions, GCP Functions)мҳҒм—ӯмқҳ Applicationл“Өмқ„ REST, gRPC, SOAP, Web Sockerкё°л°ҳмңјлЎң Aggregate н•ҙм„ң Function кё°л°ҳ м¶”мғҒнҷ”лҘј кө¬нҳ„н•ҙ мЈјлҠ” мҳӨн”ҲмҶҢмҠӨ н”„лЎңм қнҠёлқј м •мқҳ н•  мҲҳ мһҲлӢӨ.

Istioмқҳ Ingressкё°лҠҘмҷёмқҳ м—¬лҹ¬к°Җм§Җ л¶Җк°Җ кё°лҠҘ(Telemetry, Security, Policy Enforcement)л“ӨмқҖ Knativeм—җм„ңлҠ” н•„мҡ”лЎң н•ҳм§Җ м•ҠлҠ”лӢӨ.

Knative API Gateway лЎңм„ң Istioк°Җ м•„лӢҢ Glooк°Җ мЎ°кёҲлҚ” кІҪлҹүнҷ”лҗң лҢҖм•ҲмңјлЎң кІ°м •лҗҳм—Ҳкі  GlooлҘј нҶөн•ҙ Knative м„Өм№ҳк°Җ к°ҖлҠҘн•ҳкІҢ лҗҳм—ҲлӢӨ. лӢЁ, Knative Eventing м»ҙнҸ¬л„ҢнҠёлҠ” нҳ„мһ¬ м§Җмӣҗн•ҳм§Җ м•ҠлҠ”лӢӨкі  н•ңлӢӨ.

  • RouteлҠ” мӮ¬мҡ©мһҗ м„ңл№„мҠӨм—җ лҢҖн•ң HTTP endpointмҷҖ Routingмқ„ м ңкіөн•ңлӢӨ.
  • RevisionsмқҖ code(function)мҷҖ configлЎң кө¬м„ұлҗң л¶ҲліҖмқҳ мҠӨлғ…мғ·. RouteлҘј нҶөн•ҙ endpointлҘј н• лӢ№л°ӣм§Җ лӘ»н•ң RevisionмқҖ мһҗлҸҷмңјлЎң kubernetes resourceм—җм„ң мӮӯм ңлҗЁ
  • ConfigurationмқҖ мҡ”кө¬лҗҳлҠ” Revision мөңмӢ  мғҒнғңлҘј кё°лЎқн•ҳкі  мғқм„ұн•ҳкі  м¶”м Ғн• мҲҳ мһҲмқҢ. мҶҢмҠӨ нҢЁнӮӨм§Җ(git repoлӮҳ archive)лҘј м»Ён…Ңмқҙл„ҲлЎң ліҖнҷҳн•ҳкё° мң„н•ң лӮҙмҡ©мқҙлӮҳ л©”нғҖлҚ°мқҙн„°л“ұмқ„ нҸ¬н•ЁмӢңнӮ¬мҲҳ мһҲмқҢ.
  • ServiceлҠ” RoutesмҷҖ Configurations лҰ¬мҶҢмҠӨмқҳ м¶”мғҒнҷ”лҗң м§‘н•©мІҙ. лӘЁл“  мӣҢнҒ¬лЎңл“ңмқҳ lifecycleмқ„ кҙҖлҰ¬н•Ё. нҠёлһҳн”Ҫмқ„ н•ӯмғҒ мөңмӢ мқҳ RevisionмңјлЎң routeлҗҳлҸ„лЎқ м •мқҳн• мҲҳ мһҲмқҢ

н•ҳлӮҳм”© мЎ°кёҲ мһҗм„ёнһҲ мқҙм•јкё° н•ҳл©ҙ м•„лһҳмІҳлҹј м •лҰ¬ н• мҲҳ мһҲлӢӨ.

RouteвҖӢ

RouteлҠ” мӮ¬мҡ©мһҗ м„ңл№„мҠӨ(CodeмҷҖ Configurationмқҳ Revisionм •ліҙ)мқҳ л„ӨнҠёмӣҢнҒ¬ EndpointлҘј м ңкіөн•ңлӢӨ. kubernetes namespaceлҠ” м—¬лҹ¬к°ңмқҳ RouteлҘј к°Җм§ҲмҲҳ мһҲлӢӨ. RouteлҠ” н•ҳлӮҳ мқҙмғҒмқҳ Revisionsмқ„ к°Җм§Җл©ҙм„ң мҲҳлӘ…мқҙ кёёкі  м•Ҳм •м Ғмқё HTTP EndpointлҘј м ңкіөн•ңлӢӨ. кё°ліёкө¬м„ұмқҖ Route к°қмІҙк°Җ Configurationм—җ мқҳн•ҙ мғқм„ұлҗң мөңмӢ мқҳ RevisionмңјлЎң нҠёлһҳн”Ҫмқ„ мһҗлҸҷмңјлЎң м§Җм •н•ңлӢӨ. мЎ°кёҲлҚ” ліөмһЎн•ң кІҪмҡ°лЎңлҠ” istioмқҳ кё°лҠҘмқ„ нҷңмҡ©н•ҳм—¬ нҠёлһҳн”Ҫмқ„ л°ұл¶„мңЁ кё°мӨҖмңјлЎң RouteлҘј м§Җм •н•  мҲҳ мһҲлӢӨ.

RevisionвҖӢ

RevisionмқҖ CodeмҷҖ Configurationмқҳ л¶ҲліҖмқҳ мҠӨлғ…мғ·мқҙлӢӨ. н•ҳлӮҳмқҳ RevisionмқҖ м»Ён…Ңмқҙл„Ҳ мқҙлҜём§ҖлҘј м°ёмЎ°н•ҳкі  м„ нғқм ҒмңјлЎң BuildлҘј м°ёмЎ°н•  мҲҳ мһҲлӢӨ. RevisionмқҖ Configurationмқҙ м—…лҚ°мқҙнҠё мӢң мғқм„ұлҗңлӢӨ.
RouteлҘј нҶөн•ҙ httpмЈјмҶҢ м§Җм •мқҙ л¶Ҳк°ҖлҠҘн•ң RevisionмқҖ нҸҗкё° лҗҳкі  кҙҖл Ёлҗң kubernetes лҰ¬мҶҢмҠӨк°Җ мӮӯм ңк°Җ лҗңлӢӨ. мӢңк°„мқҙ м§ҖлӮЁм—җ л”°лқј Configurationмқҙ мғқм„ұн•ң Revision нһҲмҠӨнҶ лҰ¬к°Җ м ңкіөлҗҳкі  мӮ¬мҡ©мһҗлҠ” мқҙм „ RevisionлЎң мүҪкІҢ лЎӨл°ұ н•  мҲҳ мһҲлӢӨ.

ConfigurationвҖӢ

ConfigurationмқҖ мөңмӢ мқҳ RevisionмғҒнғңлҘј м„ӨлӘ…н•ҳкі , мғқм„ұн•ҳкі , мӣҗн•ҳлҠ” мғҒнғңк°Җ к°ұмӢ лҗ л•Ң Revisionмқҳ мғҒнғңлҘј м¶”м Ғн•ңлӢӨ. ConfigurationмқҖ BuildлҘј м°ёмЎ°н•ҳм—¬ мҶҢмҠӨ(git repo лҳҗлҠ” archive)лҘј м»Ён…Ңмқҙл„ҲлЎң ліҖнҷҳн•ҳлҠ” л°©лІ•м—җ лҢҖн•ң к°Җмқҙл“ңк°Җ нҸ¬н•Ёлҗҳм–ҙ мһҲкұ°лӮҳ лӢЁмҲңнһҲ м»Ён…Ңмқҙл„Ҳ мқҙлҜём§Җ л°Ҹ мҲҳм •м—җм„ң н•„мҡ”н•ң л©”нғҖ лҚ°мқҙн„° л§Ң м°ёмЎ° н•  мҲҳ мһҲлӢӨ.

  • л©ҖнӢ° нҒҙлқјмҡ°л“ң : нҚјлё”лҰӯ нҒҙлқјмҡ°л“ң to нҚјлё”лҰӯ нҒҙлқјмҡ°л“ң мҡҙмҳҒ мҶ”лЈЁм…ҳмңјлЎң н”„лқјмқҙл№— нҒҙлқјмҡ°л“ң мҳҒм—ӯм—җлҠ” кұ°мқҳ кҙҖм—¬лҘј м•Ҳн•ҳл©°, мҶ”лЈЁм…ҳм—җ л”°лқј мқјл¶Җ к°ҖмғҒнҷ”лҗң Computing мһҗмӣҗк№Ңм§Җ лӘЁлӢҲн„°л§Ғ м§Җмӣҗн•ҳкё°лҸ„ н•Ё. л©ҖнӢ° нҒҙлқјмҡ°л“ң мҡҙмҳҒ н”Ңлһ«нҸјмқҖ мЈјлЎң нҒҙлқјмҡ°л“ң Brokerage Service нҳ•нғңлЎң м ңкіөлҗҳм–ҙм•ј н•ҳлҜҖлЎң MSPм—ӯн• мқҙ мӨ‘мҡ”н•ҙм§Ҳ кІғ к°ҷлӢӨ. л©”к°ҖмЎҙ, лІ мҠӨн•ҖкёҖлЎңлІҢкіј к°ҷмқҖкіі лҝҗм•„лӢҲлқј мһҗмІҙм Ғмқё нҒҙлқјмҡ°л“ңлҘј мҡҙмҳҒн•ҳлҠ” лҢҖнҳ• SIнҡҢмӮ¬лҘј нҸ¬н•Ён•ң кё°мЎҙ IaaS мҡҙмҳҒмЎ°м§Ғмқҳ ліҖнҷ”л“ӨлҸ„ лҲҲм—¬кІЁ лҙҗм•јн• кІғ к°ҷлӢӨ. л§Ҳм§„мңЁмқҙ лӮ®мқҖ л Ҳл“ңмҳӨм…ҳ мӢёмӣҖм—җм„ң лҲ„к°Җ мқҙкё°лҠҗлғҗк°Җ мӨ‘мҡ”н•ҳкё° ліҙлӢӨлҠ” лҜёлһҳлҘј ліҙкі  нҲ¬мһҗн•ҳлҠ” мһҗм„ёк°Җ н•„мҡ”н•ҳлӢӨкі  ліёлӢӨ.

  • н”„лқјмқҙл№— нҒҙлқјмҡ°л“ң : н”„лқјмқҙл№— нҒҙлқјмҡ°л“ң м „мҡ© мҶ”лЈЁм…ҳмңјлЎң лІӨлҚ” лҳҗлҠ” нҠ№м • IaaS м„ңл№„мҠӨм—җ мөңм Ғнҷ”лҗң кІҪмҡ°к°Җ л§ҺмңјлҜҖлЎң VMware, OpenShift, OpenStack Managed м„ңл№„мҠӨл“ұ кё°мЎҙ м ңн’Ҳм—җм„ңмқҳ м—°мһҘм„ м—җм„ң м–јл§ҲлӮҳ мҳӨн”ҲмҶҢмҠӨлҘј мқҙн•ҙн•ҳкі  н”„лЎңлҚ•м…ҳм—җ м Ғмҡ©н• мҲҳ мһҲлҠ” м—ӯлҹүмқ„ кё°лҘҙлҠ”кІғмқҙ мӨ‘мҡ”н•ҳлӢӨ.

  • н•ҳмқҙлёҢлҰ¬л“ң нҒҙлқјмҡ°л“ң : нҚјлё”лҰӯ нҒҙлқјмҡ°л“ң to н”„лқјмқҙл№— нҒҙлқјмҡ°л“ң мҡҙмҳҒ мҶ”лЈЁм…ҳмңјлЎң н”„лқјмқҙл№— нҒҙлқјмҡ°л“ңмқҳ мқён”„лқјмҠӨнҠёлҹӯмІҳк№Ңм§Җ кҙҖм—¬лҘј н•ҳл©°, нҚјлё”лҰӯ нҒҙлқјмҡ°л“ң мҳҒм—ӯмқҖ мқјл°ҳм ҒмңјлЎң IaaS мҳҒм—ӯкіј мқјл¶Җ PaaS / м»Ён…Ңмқҙл„Ҳк№Ңм§Җ кө¬м„ұ, лӘЁлӢҲн„°л§Ғ м§Җмӣҗн•ҙм•ј н•ҳкі  мқҙлҘј мң„н•ң 3rd Party мҶ”лЈЁм…ҳл“Өмқҙ л“ұмһҘн•ҳкі  мһҲкі  м җм°Ё мҳҒм—ӯмқ„ нҷ•лҢҖн• кІғмқҙлӢӨ. мқҙм—җ мһҲм–ҙ м»Ён…Ңмқҙл„Ҳ н”Ңлһ«нҸјмқ„ лҸ„мһ…н•ҳлҠ”кІғмқҙ к°ҖмһҘ кёҖлЎңлІҢ н”Ңл Ҳмқҙм–ҙмҷҖ кІ©м°ЁлҘј мӨ„мқҙлҠ”лҚ° нҡЁкіјм Ғмқё лҸ„кө¬лқј мғқк°Ғн•ңлӢӨ. кІ°көӯ н•ҳмқҙлёҢлҰ¬л“ң нҒҙлқјмҡ°л“ң кө¬нҳ„мқҖ кё°мЎҙ Publicкіј Local Cloudк°„ л„ӨнҠёмӣҢнҒ¬ м—°кі„к°Җ мӨ‘мҡ”н•ң нҸ¬мқёнҠёмқҙлӢӨ.

  • к°„лӢЁнһҲ кө¬нҳ„н•ҙ ліјмҲҳ мһҲлҠ” General Hybrid Kubernetes ArchitectureлҘј к°„лӢЁнһҲ к·ёл ӨлҙӨлӢӨ. кё°ліём ҒмңјлЎң StrongSwan VPNмқ„ к°Җм§Җкі  CI/CDлҘј кі л Өн•ң DevOpsкҙҖм җм—җм„ңмқҳ кө¬м„ұлҸ„мқҙлӢӨ.

мң„м—җ лӮҙмҡ©кіј н•Ёк»ҳ н•ҳмқҙлёҢлҰ¬л“ң нҒҙлқјмҡ°л“ңлҘј кі л Өн• л•Ң лҚ°мқҙн„° мҶҢмң к¶Ң мёЎл©ҙм—җм„ңмқҳ кұ°лІ„л„ҢмҠӨ мҷҖ overlay л„ӨнҠёмӣҢнҒ¬м—җ лҢҖн•ң кі лҜјмқҙ лҸҷн–үлҗҳм—Ҳмқ„л•Ң м„ұкіөм Ғмқё н•ҳмқҙлёҢлҰ¬л“ң нҒҙлқјмҡ°л“ңлҘј кө¬м¶•н• мҲҳ мһҲлӢӨлқјлҠ” лӮҳ нҳјмһҗл§Ңмқҳ мғқк°Ғмқ„ м •лҰ¬н•ҙлҙӨлӢӨ.

SRE (Site Reliablity Engineering)

January 30, 2019 В· 5 min read

к·ёлҸҷм•Ҳ DevOps лӢҙлӢ№мһҗлқјкі  л¶ҖлҘҙм§–лҚҳ мӮ¬лһҢл“Өмқҳ Roleмқҙ мқён”„лқј лӢҙлӢ№мһҗмқём§Җ н”Ңлһ«нҸј лӢҙлӢ№мһҗмқём§Җ м•„лӢҲл©ҙ к°ңл°ңн•ҳлҠ” мҡҙмҳҒмһҗмқём§Җ н—·к°ҲлҰҙл•Ңк°Җ л§Һм•ҳлӢӨ.

к°‘мһҗкё° көӯлӮҙм—җм„ңлҸ„ SRE мұ„мҡ© кіөкі к°Җ л§Һм•„м§ҖлҠ”кІғмқ„ ліҙл©ҙм„ң лӮҙ мһҗмӢ лҸ„ н•ңлІҲ м •лҰ¬лҘј н•ҳкі  к°Җм•јн• кІғ к°ҷмқҖ мғқк°Ғмқҙ л“Өм—ҲлӢӨ.

https://docs.microsoft.com/ko-kr/learn/modules/intro-to-site-reliability-engineering/

к°ңл…җм •лҰ¬ мёЎл©ҙм—җм„ң мң„ MS Azureмқҳ мҳЁлқјмқё көҗмңЎлӮҙмҡ©мқ„ м •лҰ¬н•ҙлҙӨлӢӨ.

SREвҖӢ

SRE(мӮ¬мқҙнҠё м•Ҳм •м„ұ м—”м§ҖлӢҲм–ҙл§Ғ)лһҖ мЎ°м§Ғмқҙ н•ҙлӢ№ мӢңмҠӨн…ң, м„ңл№„мҠӨ л°Ҹ м ңн’Ҳм—җм„ң м Ғм Ҳн•ң мҲҳмӨҖмқҳ м•Ҳм •м„ұмқ„ лӢ¬м„ұн•ҳлҸ„лЎқ м§Җмӣҗн•ҳлҠ” м—”м§ҖлӢҲм–ҙл§Ғ л¶„м•јмқҙлӢӨ.

м¶ңмІҳ : https://landing.google.com/sre/sre-book/chapters/introduction/

кё°ліём ҒмңјлЎң мҡҙмҳҒкІҪн—ҳмқҙ мһҲлҠ” sysadmin, ITм „л¬ёк°Җ, DevOpsмӢӨн–ү лӢҙлӢ№мһҗ л“ұмқҙ лҢҖмғҒмқҙ лҗ  мҲҳ мһҲлӢӨ.

м Ғм Ҳн•ң мҲҳмӨҖмқҳ м•Ҳм •м„ұвҖӢ

к°ҖмһҘ н•өмӢ¬мқҙ лҗҳлҠ” лӢЁм–ҙлҠ” м•Ҳм •м„ұмқҙлӢӨ. мЎ°м§Ғм—җм„ң л§Ңл“  м• н”ҢлҰ¬мјҖмқҙм…ҳмқҙ м•Ҳм •м„ұмқҙ л¶ҖмЎұн•ҳм—¬ мһ‘лҸҷн•ҳм§Җ м•Ҡкұ°лӮҳ л¶Ҳм•Ҳм •н•ң кІҪмҡ° мӢӨм ң л№„мҰҲлӢҲмҠӨм—җ н”јн•ҙлҘј мӨ„ мҲҳ мһҲлӢӨлҠ”кұҙ лҲ„кө¬лӮҳ м•Ңкі  мһҲлҠ” лӢ№м—°н•ң мқҙм•јкё°мқҙм§Җл§Ң 100%мқҳ м•Ҳм •м Ғмқё мӢңмҠӨн…ңмқҖ мЎҙмһ¬н•ҳм§Җ м•ҠмңјлҜҖлЎң м Ғм Ҳн•ң мҲҳмӨҖ(м •лҹүм ҒмңјлЎң мёЎм •н•ҳкёҙ м–ҙл өм§Җл§Ң мқҙн•ҙкҙҖкі„мһҗл“Өмқҙ лӮ©л“Өн• л§Ңн•ң мҲҳмӨҖ)мқҳ м•Ҳм •м„ұмқ„ лӢ¬м„ұн•ҳкё° мң„н•ң мқјл Ёмқҳ мһ‘м—…л“Өмқ„ мҲҳн–үн•ҳлҠ” кІғмқҙ SREмқҳ кё°ліё мҶҚм„ұмқҙлқјкі  ліјмҲҳ мһҲлӢӨ.

SREлҠ” Googleм—җм„ң 2003л…„л¶Җн„° 7лӘ…мқҳ мҶҢн”„нҠёмӣЁм–ҙ м—”м§ҖлӢҲм–ҙлЎң кө¬м„ұлҗң нҢҖм—җм„ң мӢңмһ‘лҗң мқҙнӣ„ (мһҗм„ён•ң лӮҙмҡ©мқҖ мң„ л§ҒнҒ¬ ebook м°ёмЎ°) Google мӮ¬лӮҙм—җ л„җлҰ¬ нҷ•мӮ°лҗҳкі  лӮҙл¶Җм Ғмқё л¬ёнҷ”лЎң мЎ°м„ұлҗҳлҠ” мӨ‘м—җ л°–м—җм„ңлҠ” DevOpsлқјкі  н•ҳлҠ” л¬ёнҷ”к°Җ лҸҷмӢңм—җ нҷ•мӮ°лҗҳм—ҲлӢӨ н•ңлӢӨ. кІ°көӯ лҸҷмқјн•ң л¬ём ңлҘј н•ҙкІ°н•ҳкё° мң„н•ң мёЎл©ҙм—җм„ң л‘җк°ңмқҳ л°©лІ•лЎ мқҖ лӢӨлҘҙлӢӨкі  лҙҗм•јн•ңлӢӨ.

м°Ёмқҙм җвҖӢ

  • SRE - м•Ҳм •м„ұмқ„ мң„н•ң м—”м§ҖлӢҲм–ҙл§Ғ, DevOps - к°ңл°ңкіј мҡҙмҳҒмЎ°м§Ғ к°Ғк°Ғмқҳ мӮ¬мқјлЎңлҘј н•ҙмІҙн•ҳкё° мң„н•ң л¬ёнҷ”м Ғмқё мӣҖм§Ғмһ„
  • SRE - м ҖлҠ” SRE мһ…лӢҲлӢӨ, DevOps - м ҖлҠ” DevOpsлҘј н•ҳлҠ” мҡҙмҳҒмһҗ лҳҗлҠ” к°ңл°ңмһҗ мһ…лӢҲлӢӨ.
  • SRE - к·ңлІ”мңјлЎң мқёмӢқ, DevOps - л¬ёнҷ”лЎң мқёмӢқ

кіөнҶөм җвҖӢ

  • лӘЁлӢҲн„°л§Ғ/мӢқлі„ к°ҖлҠҘ, мһҗлҸҷнҷ”

мЈјмҡ” SREмӣҗм№ҷ : м„ мҲңнҷҳвҖӢ

мғҲлЎңмҡҙ м„ңл№„мҠӨмқҳ мғҒнғңмқҳ м§Җн‘ңлҠ” м–ҙл–»кІҢ м •мқҳн• к№Ң? ліҙнҶөмқҖ л¬ҙм—Үмқ„ SLI(м„ңл№„мҠӨ мҲҳмӨҖ м§Җн‘ң)лЎң мӮ¬мҡ©н• м§ҖлҘј м •н•ҳлҠ”кІғмңјлЎң ліёлӢӨ. мқјл°ҳм ҒмңјлЎң м„ұкіө л°Ҹ мӢӨнҢЁ мёЎм •к°’(200 OK,500 Error), мқ‘лӢөмӢңк°„(ms), мІҳлҰ¬лҹү л“ұмқҳ мЎ°н•©мңјлЎң кІ°м •лҗ  мҲҳ мһҲлӢӨ. ліҙнҶө м№ҙлӮҳлҰ¬м•„ л¶„м„қмқ„ нҶөн•ҙ Scoringмқ„ н•ң SLIлҘј 200 OKмҷҖ 500 лҳҗлҠ” м—җлҹ¬мҪ”л“ңмқҳ л№„мңЁлЎң кІ°м •н•ҳлҠ” л°©лІ•мқ„ мӮ¬мҡ©н•ҳлҠ” кІҪмҡ°к°Җ л§ҺлӢӨ.

м„ңл№„мҠӨ мғҒнғңлҘј мӢқлі„н•ҳлҠ” м§Җн‘ңлҘј м •н•ҳкі  лӮҳл©ҙ кі к°қмқҙлӮҳ мҡ°лҰ¬к°Җ мӣҗн•ҳлҠ” м•Ҳм •м„ұ мҲҳмӨҖмқ„ кІ°м •н•ҙм•ј н•ңлӢӨ. к°ңл°ңнҢҖкіј мҡҙмҳҒнҢҖмқҙ к°ҷмқҙ л§Ңл“  мӣҗн•ҳлҠ” м•Ҳм „м„ұ мҲҳмӨҖмқ„ SLO(м„ңл№„мҠӨ мҲҳмӨҖ лӘ©н‘ң)лқјкі  л§җн•ҳл©ҙ лҗңлӢӨ.

SLOлҠ” PrometheusлӮҳ Datadogкіј к°ҷмқҖ лӘЁлӢҲн„°л§Ғ мӢңмҠӨн…ңмқ„ нҷңмҡ©н•ҳм—¬ м •нҷ•н•ҳкІҢ мёЎм •н•ҳкі  н‘ңмӢңн•ҙм•ј н•ңлӢӨ. м„ңл№„мҠӨмқҳ м•Ҳм •м„ұм—җ лҢҖн•ҙм„ң лӘ…нҷ•н•ҳкІҢ мқҙн•ҙн• мҲҳ мһҲлҠ” лӘ©н‘ңмқҙкё°лҸ„ н•ҳлӢӨ. л°ҳл“ңмӢң SLOлҘј л§ҢмЎұн•ҳлҠ”м§Җ лӘ»н•ҳлҠ”м§Җ лӘ…нҷ•н•ң мёЎм •к°’мңјлЎң лҚ°мқҙн„°к°Җ мЎҙмһ¬н•ҙм•ј н•ҳкі  SLOлҘј м¶©мЎұн•ҳм§Җ лӘ»н•ҳл©ҙ л¬ём ңк°Җ мһҲлҠ”кІғмңјлЎң нҢҗлӢЁн•ҳкі  л¬ём ңлҘј н•ҙкІ°н•ҙм•ј н•ңлӢӨ.

мҳӨлҘҳ мҳҲмӮ°вҖӢ

мҳӨлҘҳ мҳҲмӮ°мқҙлқјлҠ” л§җмқ„ лӘ…нҷ•нһҲ мқҙн•ҙн•ҳл Өл©ҙ нқ”нһҲ кі к°қкіј мІҙкІ°н•ҳлҠ” кі„м•Ҫм„ң мғҒмқҳ лӘ…мӢңлҗң м„ңл№„мҠӨ к°ҖлҸҷмңЁ(SLA)мқ„ мғқк°Ғн•ҳл©ҙ лҗ  кІғ к°ҷлӢӨ. ліҙнҶө 99.9% мқҳ SLOлҘј м •н•ңлӢӨкі  н•ңлӢӨл©ҙ 1л…„м—җ 8мӢңк°„ 45л¶„ 57мҙҲмқҳ м„ңл№„мҠӨ лӢӨмҡҙнғҖмһ„мқ„ н—Ҳмҡ©н•ҳлҠ” кІғмқҙлӢӨ. мҳӨлҘҳ мҳҲмӮ°мқҖ м„ңл№„мҠӨмқҳ мҷ„лІҪн•ң м•Ҳм •м„ұкіј мӣҗн•ҳлҠ” м•Ҳм •м„ұмқҳ м°Ёмқҙ(100%-99.9%=0.01%)лҘј л§җн•ңлӢӨ. 0.01%лЎң мҰү 8мӢңк°„м •лҸ„мқҳ мҳӨлҘҳ мҳҲмӮ°мқ„ к°Җм§ҖлҠ” кІғмқҖ к·ё мӢңк°„мқ„ лӘЁл‘җ мӮ¬мҡ©н•  л•Ңк№Ңм§Җ м¶”к°Җ лҰҙлҰ¬мҠӨлӮҳ нҢЁм№ҳлҘј м§„н–үн•  мҲҳ мһҲлӢӨлҠ” мқҙм•јкё°мҷҖ к°ҷлӢӨ. м–ҙл–Ө нҢҖмқҖ н•ҙлӢ№ мҳҲмӮ°мқ„ мӢ к·ң кё°лҠҘмқ„ лҰҙлҰ¬мҠӨн•ҳлҠ”лҚ° мӮ¬мҡ©н•ҳкё°лҸ„ н•ҳкі  мһҘм• к°Җ л°ңмғқн–Ҳмқ„л•Ң л¬ём ңмқҳ мӣҗмқёмқ„ л°ңкІ¬н•ҳкі  к°ңм„ н•ҳлҠ”лҚ° мӮ¬мҡ©н•ҳкё°лҸ„ н•ңлӢӨ.

мқјл°ҳм ҒмңјлЎң м„ңл№„мҠӨм—җ лҢҖн•ң мҳӨлҘҳ мҳҲмӮ°мқҙ лӘЁл‘җ мӮ¬мҡ©лҗҳл©ҙ м„ңл№„мҠӨк°Җ мӣҗн•ҳлҠ” м•Ҳм •м„ұ мҲҳмӨҖмңјлЎң лҸҢм•„к°Ҳл•Ң к№Ңм§Җ н•ҙлӢ№ м„ңл№„мҠӨмқҳ м¶”к°Җм Ғмқё лҰҙлҰ¬мҠӨлҘј ліҙлҘҳн•ҳлҠ”кІҢ мқјл°ҳм Ғмқҙкёҙ н•ҳм§Җл§Ң нҠ№м •кё°к°„(мӣ” лҳҗлҠ” л¶„кё° лҳҗлҠ” м—°к°„) кё°мӨҖмңјлЎң кі„мӮ°лҗҳкё° л•Ңл¬ём—җ м„ңл№„мҠӨ м•Ҳм •м„ұмқҙ м •мғҒ мғҒнғңлқјл©ҙ мҳӨлҘҳ мҳҲмӮ°мқҖ лӢӨмӢң мЈјм–ҙм§ҖкІҢ лҗҳлҜҖлЎң м„ мҲңнҷҳ кө¬мЎ°лҘј к°Җм§„лӢӨкі  ліј мҲҳ мһҲлӢӨ.

л№„лӮң м—ҶлҠ” мһҘм•  нҡҢкі вҖӢ

мқјл°ҳм ҒмңјлЎң л№„мҰҲлӢҲмҠӨм—җ нҒ¬лҰ¬нӢ°м»¬н•ң мһҘм• к°Җ л°ңмғқн–Ҳмқ„л•Ң мӮ¬нӣ„ л¶„м„қ(нҡҢкі )мқ„ н•ҳкІҢ лҗҳлҠ”лҚ° мқҙл•ҢлҠ” л№„лӮң м—ҶлҠ” л¶„м„қмқ„ н•ҙм•јн•ңлӢӨ. мҳӨлһҳм „ кё°м–өмқҙм§Җл§Ң мҳҲм „ м§ҒмһҘ нҠ№м •нҢҖм—җм„ңлҠ” мһҘм• к°Җ л°ңмғқн•ҳл©ҙ мң кҙҖмһҗл“Өмқҙ лӘЁл‘җ м—”м§ҖлӢҲм–ҙ мұ…мғҒ л’Өм—җ м„ңм„ң лӘЁл“  нҷ”мӮҙкіј лҲҲмҙқмқ„ мЈјкё° л•Ңл¬ём—җ лҸ„м ҖнһҲ мқјмқ„ н•  мҲҳ м—ҶкІҢ л§Ңл“  кІҪмҡ°к°Җ л§Һм•ҳм—ҲлӢӨ.

нҠ№м • мҡҙмҳҒмһҗмқҳ мһ‘м—… мӢӨмҲҳлЎң мқён•ҙ мһҘм• к°Җ л°ңмғқн–ҲлӢӨ н•ҳлҚ”лқјлҸ„ н•ҙлӢ№ мқҙлІӨнҠёк°Җ л°ңмғқн•ҳкІҢ лҗң н”„лЎңм„ёмҠӨлӮҳ кё°мҲ кё°л°ҳм—җ мӢӨнҢЁ мӣҗмқёмқ„ нҢҢм•…н•ҳлҠ”лҚ° мӨ‘м җмқ„ л‘җм–ҙм•ј н• кІғмқҙлӢӨ.

мӢңмҠӨн…ңмқҳ м•Ҳм •м„ұмқ„ м Җн•ҳмӢңнӮӨкі  м„ңл№„мҠӨлҘј мӨ‘лӢЁмӢңнӮЁ мһ‘м—…(лҰҙлҰ¬мҠӨ, нҢЁм№ҳ л“ұ)мқ„ мӮ¬мң лЎң к°ңмқём—җкІҢ нҺҳл„җнӢ°лҘј мЈјкұ°лӮҳ кі кіјм—җ л°ҳмҳҒн•ҳлҠ” л°©лІ•мқҖ мһҘм• лЎң л¶Җн„° көҗнӣҲмқ„ м–»мқ„мҲҳк°Җ м—Ҷкі  н•ҙлӢ№ лӢҙлӢ№мһҗмқҳ м¶©м„ұлҸ„лӮҳ мғқмӮ°м„ұ м Җн•ҳлҘј мң л°ңн•ҳлҠ” м•ҲмўӢмқҖ мһҘм№ҳк°Җ лҗ  мҲҳ мһҲлӢӨ. лӮҙ кІҪн—ҳмқҙкё°лҸ„ н•ҳм§Җл§Ң н•ңлҸҷм•Ҳ л‘җл ӨмӣҢм„ң м•„л¬ҙ ліҖкІҪлҸ„ н•ҳм§Җ лӘ»н•ҳлҠ” кІҪмҡ°к°Җ мў…мў… мһҲм—ҲлӢӨ.

нҡҢмӮ¬лӮҳ мЎ°м§Ғ, нҢҖмқҖ мӢңмҠӨн…ң мӨ‘лӢЁмңјлЎңл¶Җн„° көҗнӣҲмқ„ м–»кі  м§ҖмҶҚм ҒмңјлЎң мӢңмҠӨн…ңмқ„ к°ңм„ н• мҲҳ мһҲлӢӨ. м Ғм Ҳн•ң л¶„м„қмқ„ нҶөн•ҙ нӣ„мҶҚмЎ°м№ҳлҘј мҲҳн–үн•ЁмңјлЎңмҚЁ мӢӨнҢЁлҘј мҲҳмҡ©н•ҳлҠ” кІғмқҙ SREмқҳ н•өмӢ¬ мӣҗм№ҷмқҙлӢӨ. мқҙлҹ¬н•ң лӮҙмҡ©мқ„ лӘЁл‘җм—җкІҢ кіөмң н•ҳкі  нғҖ мЎ°м§ҒмқҙлӮҳ нҢҖм—җкІҢ мқёмӮ¬мқҙнҠёлҘј м ңкіөн•ҳлҠ” м„ мҲңнҷҳ кө¬мЎ°лҘј л§Ңл“ңлҠ”кІғ лҳҗн•ң SREмқҳ кё°ліё мӣҗм№ҷмқҙлқјкі  ліј мҲҳ мһҲлӢӨ.

мЈјмҡ” SREмӣҗм№ҷ : мӮ¬лһҢмёЎл©ҙвҖӢ

Toil(мҲҳкі мҠӨлҹ° мқј, лІҲкұ°лЎңмҡҙ мһ‘м—…)вҖӢ

н•ӯмғҒ SREм—җм„ңлҠ” Toilм—җ мҙҲм җмқ„ л§һм¶°м•ј н•ңлӢӨ. ToilмқҖ мӮ¬лһҢмқҙ мҲҳн–үн•ҳлҠ” мҡҙмҳҒ мһ‘м—… мӨ‘ нҠ№м •н•ң нҢЁн„ҙмқҙлӮҳ нҠ№м„ұмқҙ мһҲлҠ” мһ‘м—…мқ„ л§җн•ңлӢӨ. мў…мў… л°ҳліөм Ғмқҙкі  мһҗлҸҷнҷ” лҗ  мҲҳ мһҲлҠ” мһ‘м—…мқјм§ҖлқјлҸ„ лҢҖл¶Җл¶„ мҲҳлҸҷмһ‘м—…мңјлЎң мІҳлҰ¬лҘј н•ңлӢӨ. м„ңл№„мҠӨмқҳ к·ңлӘЁк°Җ м»Өм§Җкұ°лӮҳ кі к°қмқҙ л§Һм•„м§ҖкІҢ лҗҳл©ҙ ліҙнҶө мҡ°лҰ¬к°Җ мқҙм•јкё° н•ҳлҠ” нӢ°мј“(лҳҗлҠ” SR)мқҳ мҲҳк°Җ л§Һм•„м§Җкі  кіөмҲҳк°Җ л§Һмқҙ н•„мҡ”н•ҳкІҢ лҗңлӢӨ.

л°°нҸ¬л•Ңл§ҲлӢӨ ConfigлҘј м Ғмҡ©н•ҳкұ°лӮҳ кі„м • л“ұлЎқ, ліјлҘЁмқҙлӮҳ л””мҠӨнҒ¬ н”„лЎңл№„м ҖлӢқ л“ұ мҲҳлҸҷмңјлЎң мІҳлҰ¬н•ҙм•јн•ҳлҠ” лӘЁл“  мһ‘м—…мқҙ мҡҙмҳҒмһҗм—җкІҢлҠ” л¶Җн•ҳлЎң лӢӨк°Җмҳ¬мҲҳ мһҲлӢӨ. мқҙлҹ¬н•ң л°ҳліөм Ғмқё мһ‘м—…мқ„ нӢ°мј“мӢңмҠӨн…ңмқ„ нҶөн•ҙ мІҳлҰ¬лҘј н•ҳкІҢ лҗҳл©ҙ нӢ°мј“ мғқм„ұ, мһ‘м—…кі„нҡҚм„ң мһ‘м„ұ, кІҖнҶ , мҠ№мқё лӢЁкі„лҘј к·ёл•Ң л§ҲлӢӨ н•ҙм•јн•ҳлҠ” м•„мЈј лІҲкұ°лЎңмҡҙ мһ‘м—…(Toil)мқ„ н•ҙм•јн•ңлӢӨ.

SREлҠ” мөңлҢҖн•ң мқҙлҹ¬н•ң л°ҳліөм Ғмқҙкі  лІҲкұ°лЎңмҡҙ мһ‘м—…мқ„ м ңкұ°н•ҳкё° мң„н•ҙ л…ёл Ҙн•ҙм•јн•ҳкі  мһҗлҸҷнҷ”мҷҖ Self-ServiceлЎң к°ңл°ңмһҗк°Җ нҺён•ҳкІҢ м§Ғм ‘ мӮ¬мҡ©н• мҲҳ мһҲлҸ„лЎқ нҷҳкІҪмқ„ м ңкіөн•ҙ мЈјм–ҙм•ј н•ңлӢӨ. мқҙлҹ¬н•ң мҡ”мІӯмқҙлӮҳ нӢ°мј“мқ„ мһҗлҸҷмңјлЎң мІҳлҰ¬н•  мҲҳ мһҲмңјл©ҙ мЎ°кёҲ лҚ” мғқмӮ°м Ғмқё мқјмқ„ мҲҳн–үн• мҲҳ мһҲлӢӨ.

мң„м—җм„ң мқҙм•јкё°н•ң м Ғм Ҳн•ң м•Ҳм •м„ұкіј л№„мҠ·н•ң л§ҘлқҪмңјлЎң мғқк°Ғн•ҳл©ҙ лҗ  кІғ к°ҷлӢӨ. лІҲкұ°лЎңмҡҙ мһ‘м—…мқ„ м—Ҷм• лҠ”кІғмқҙ лӢӨлҘё мӨ‘мҡ”н•ң мһ‘м—…ліҙлӢӨ мҡ°м„ мҲңмң„к°Җ л–Ём–ҙм§Җкё°лҸ„ н•ҳм§Җл§Ң м„ңл№„мҠӨ мғҒм—җм„ң Toilмқ„ м ңкұ°н•ҳлҠ”кІғлҸ„ SREмқҳ мЈјлҗң м—…л¬ҙмқҙлӢӨ.

мҡҙмҳҒмһ‘м—…мқҳ л№„мңЁвҖӢ

Toilмқ„ м ңкұ°н•ҳкұ°лӮҳ м„ңл№„мҠӨ/мӢңмҠӨн…ң м•Ҳм •м„ұмқ„ к°ңм„ н•ҳкё° мң„н•ҙ SREлҠ” мһҘм• лҘј н•ҙкІ°н•ҳкі  нӢ°мј“мқ„ мІҳлҰ¬н•ҳлҠ” мӢңк°„ л№„мӨ‘мқ„ 50%лҘј л„ҳкІЁм„ңлҠ” м•ҲлҗңлӢӨ. нӢ°мј“мқҙ н•„мҡ”н•ҳм§Җ м•ҠлҸ„лЎқ к°ңл°ңмһҗ Self-ServiceлҘј кө¬м„ұ/м ңкіөн•ҳкі  нҡЁмңЁм„ұмқ„ мҰқлҢҖмӢңнӮӨкё° мң„н•ҙм„ң IaCл“ұ(Shell, Ansible, Terraform)кіј к°ҷмқҙ мһҗлҸҷнҷ”лҘј мң„н•ң мҪ”л“ңлҘј л§Ңл“ңлҠ”лҚ°м—җлҸ„ м§‘мӨ‘мқ„ н•ҙм•јн•ңлӢӨ.

м •лҰ¬вҖӢ

м§ҖкёҲк№Ңм§Җ SREм—җ лҢҖн•ҙм„ң к°„лӢЁн•ҳкІҢ мҶҢк°ңн•ҳкі  мқҙн•ҙн•ҳлҠ” мёЎл©ҙм—җм„ң м •лҰ¬н•ҙ лҙӨлӢӨ.

к·ёлҰјкіј к°ҷмқҙ м„ңл№„мҠӨ м•Ҳм •м„ұ кі„мёө кө¬мЎ°м—җм„ңлҠ” кё°ліём ҒмңјлЎң лӘЁл“  м„ңл№„мҠӨлҘј лӘЁлӢҲн„°л§Ғ н•ҳлҠ” мӢңмҠӨн…ңмқ„ к°–м¶”м–ҙм•ј н•ңлӢӨкі  л§җн•ңлӢӨ. лӘЁлӢҲн„°л§Ғ мӢңмҠӨн…ңмқҙ мӨҖл№„к°Җ лҗҳл©ҙ м„ңл№„мҠӨм—җ лҢҖн•ң SLI, SLOлҘј л§Ңл“Өкі  лӘЁлӢҲн„°л§Ғ мӢңмҠӨн…ңм—җм„ң кө¬нҳ„н•ҳлҠ”кІғмқҙ мІ«лІҲм§ё SREлҘј м Ғмҡ©н•ҳлҠ” л°©лІ•мқҙлқјкі  н•  мҲҳ мһҲмқ„ кІғмқҙлӢӨ.
